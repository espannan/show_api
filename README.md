Show API
========

A simple API to store shows and venues.

Requirements
------------

If you are first setting up, run:

```bash
$ mkvirtualenv show_api
$ pip install -r requirements.txt
```

If returning and updating requirements, run:

```bash
$ workon show_api
$ pip install -r requirements.txt
```

Running the server
------------------

```
$ ./manage.py runserver
```
